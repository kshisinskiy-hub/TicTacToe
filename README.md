# Крестики-нолики (TicTacToe)

Простая игра «Крестики-нолики» на языке C# с использованием технологии WinForms.

## Возможности
- Игра для двух игроков на одном компьютере (X и O).
- Определение победителя и ничьей.
- Подсветка победной комбинации.
- Сохранение и загрузка состояния игры в файл `.ttt`.
- Кнопка «Новая игра» для перезапуска.

## Запуск
1. Открыть `TicTacToe.sln` в Visual Studio 2022/2026.
2. Нажать **F5** (запуск с отладкой) или **Ctrl+F5** (без отладки).

## Требования
- .NET 8.0 (Windows)
- Visual Studio 2022 или новее
